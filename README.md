# Food Recommendation Chatbot

## Overview
An academic project developed using the Scala programming language.  
A command-line chatbot that recommends restaurants based on user preferences such as cuisine, city, budget, and vibe.

The chatbot processes user input, manages preferences, stores conversation history, detects mood, and provides personalized recommendations.

## Features
- Restaurant recommendation system
- Keyword-based input processing
- User preference management
- Conversation history tracking
- Mood detection and topic analysis

## Technologies Used
- Scala
- Pattern Matching
- Case Classes
- Higher-Order Functions
- Object-Oriented Programming


## Project Structure
- `Main.scala` → Starts the application
- `Chatbot.scala` → Handles chatbot logic and responses
- `InputHandler.scala` → Processes user input
- `User.scala` → Manages user preferences
- `ChatHistory.scala` → Stores and analyzes conversations
- `FoodPlace.scala` → Handles restaurant data
